Introduction

RoastMaster AI is a generative AI chatbot designed to roast users humorously and contextually. Built on local LLMs and semantic search, it delivers wit, sarcasm, and personality.

System Workflow

  • User Input: The user types something roast-worthy.
  • Semantic Search: Finds similar examples using MiniLM embeddings.
  • Prompt Building: Combines user input + relevant examples.
  • LLM Response: Generated via Mistral 7B using LM Studio API.
  • UI Output: Roast shown on Streamlit interface with retry options.

Model Architecture

Mistral-7B-Instruct-v0.3 is used for roast generation, fine-tuned on humorous, sarcastic interactions.

  • 7B parameters, instruction-tuned
  • Context-aware and witty responses

MiniLM-L6-v2 powers the semantic search for roast prompts.

Sample Code

from openai import OpenAI
from roast_engine import get_similar_examples, build_prompt

client = OpenAI(base_url=\"http://127.0.0.1:1234/v1\", api_key=\"lm-studio\")
MODEL = \"mistral-7b-instruct-v0.3\"

user_input = \"I think I'm the funniest person alive\"
examples = get_similar_examples(user_input)
prompt = build_prompt(examples, user_input, username=\"Ajay\")

response = client.chat.completions.create(
    model=MODEL,
    messages=[{\"role\": \"user\", \"content\": prompt}],
    temperature=0.85,
    max_tokens=150
)

print(\"RoastMaster says:\", response.choices[0].message.content.strip())
